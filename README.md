# Agentic AI Learning Materials

This repository contains resources and materials for learning Generative AI. It is designed to help learners understand the concepts, tools, and techniques required to work with Generative AI effectively.

## Contents

- **Week 1**: Introduction to AI
- **Week 2**: Python Basics
- **Week 3**: Pandas and NumPy
- **Week 4**: Introduction to Generative AI
- Additional resources, assignments, and quizzes for hands-on learning.

## Purpose

The goal of this repository is to provide a structured learning path for individuals interested in exploring the field of Generative AI. It includes theoretical explanations, practical examples, and exercises to reinforce learning.

## How to Use

1. Navigate through the weekly folders to access the materials.
2. Follow the instructions provided in each week's content.
3. Complete the assignments and quizzes to test your understanding.
4. Refer to the additional resources for deeper insights.

Happy Learning!
